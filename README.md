# Student Result Management System

## Overview

A web-based Student Result Management System developed using Python and Flask. The application allows users to enter student marks and automatically generates academic results.

## Features

* Calculate Total Marks
* Calculate Percentage
* Find Maximum and Minimum Marks
* Automatic Grade Generation
* Pass/Fail Evaluation
* Responsive Bootstrap User Interface

## Technologies Used

* Python
* Flask
* HTML
* Bootstrap 5

## Installation

1. Clone the repository

2. Install dependencies

pip install -r requirements.txt

3. Run the application

python app.py

4. Open in browser

http://127.0.0.1:5000
